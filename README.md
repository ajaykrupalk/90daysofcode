<h3>N-Queens Problem</h3>
The problem statement is that no two queens should be under attack.<br>
Two queens are under attack if they are in:<br>
1. Same row<br>
2. Same column<br>
3. Same diagonal<br>
The solution for a 8*8 matrix is as follows: [https://en.wikipedia.org/wiki/Eight_queens_puzzle]<br>

<h3>Knights Tour Problem</h3>
The problem statement is Given a N*N board with the Knight placed on the first block of an empty board. Moving according to the rules of chess knight must visit each square exactly once. Print the order of each cell in which they are visited.<br>
Reference: [https://www.geeksforgeeks.org/the-knights-tour-problem-backtracking-1/?ref=lbp]<br>
The solution for a 8*8 matrix is as follows: [https://upload.wikimedia.org/wikipedia/commons/thumb/d/da/Knight%27s_tour_anim_2.gif/375px-Knight%27s_tour_anim_2.gif]
